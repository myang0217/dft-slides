# 十页读懂 DFT

一份用 [reveal.js](https://revealjs.com) 做的幻灯片,10 页讲清密度泛函理论的主线逻辑。

**在线观看:** https://myang0217.github.io/dft-slides/

横版画布 1280 × 720(投影仪比例),内容以两栏为主。

## 内容主线

| 页 | 主题 |
|---|---|
| 1 | 封面 |
| 2 | 问题:多体薛定谔方程与「指数墙」 |
| 3 | Hohenberg–Kohn 定理:$3N$ 维塌缩到 $3$ 维 |
| 4 | Kohn–Sham 映射:把未知全部扫进 $E_{\text{xc}}$ |
| 5 | Kohn–Sham 方程与自洽循环 (SCF) |
| 6 | 交换关联泛函:LDA → GGA → meta-GGA → 杂化 |
| 7 | 落地:基组、赝势、k 点采样与收敛测试 |
| 8 | 能算什么:结构、能带、声子,直到 TDDFT / GW-BSE |
| 9 | 必须知道的坑:带隙、范德华、强关联、自相互作用 |
| 10 | 一句话总结 |

面向刚接触 DFT 的物理 / 化学 / 材料背景听众:保留关键公式,不做推导。

## 技术要点

画布尺寸就是配置里的两个数字:

```js
Reveal.initialize({
  width:  1280,   // 16:9 投影仪比例
  height: 720,
});
```

公式由 KaTeX 插件渲染,直接在正文里写 `$...$` / `$$...$$`。

## 用法

- **改内容**:编辑 `index.html`,每页是一个 `<section>`;提交后 GitHub Pages 约 1 分钟自动更新。
- **换主题**:改 `<head>` 里 `theme/white.css` 一行。
- **导出 PDF**:网址后加 `?print-pdf`,浏览器打印 → 另存为 PDF(已处理脱壳规则)。
- **演讲者视图**:按 `S`,可看备注与计时。
